Steps to run the code:

Extract the project folder.

The project is build on Java and MySQL.

Eclipse or any other java compatible software should be installed. Java connector should be downloaded and externally referred.

MAMP Server should be installed to run the project.

contact.sql is the database file set up according to the given schema which is converted file from contact.csv file.

Copy Library folder into MAMP/htdocs/ folder.

After that open MAMP to enable apache and Mysql server.

Data for database can be imported from "contact.sql" file in phypMyAdmin (MAMP).

Internet is needed for proper working of project.


Libraries and Software Used 
Software Used: Mamp(Apache, MySQL, php), Eclipse.
OS : Windows 10
Database: MySQL 
Language: Java, Java Swing, JDBC connector.
